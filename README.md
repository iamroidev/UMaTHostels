## Overview

Expo mobile application for UMaT students to discover, compare, and book hostels around Tarkwa.

---

## 🛠️ Technology Stack & Details

*   **Framework**: React Native with Expo SDK (TypeScript).
*   **Routing**: React Navigation or Expo Router.
*   **State / Storage**: AsyncStorage or state managers.

## 🚀 Getting Started

### Prerequisites

Ensure you have the required runtimes installed:
*   [Node.js](https://nodejs.org/) (v18.0.0 or higher)
*   [Git](https://git-scm.com/)
*   Expo Go app on your mobile device (iOS/Android)

### Setup Instructions

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/iamroidev/UMaTHostels.git
    cd UMaTHostels
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Launch Application**
    ```bash
    npx expo start
    ```
